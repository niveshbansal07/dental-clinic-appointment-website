Use the secure form below to request an appointment with our dentists. Our team will confirm the nearest available slot via WhatsApp, SMS or email. For immediate pain or swelling, please call our emergency line.

Appointment details

You will also receive a confirmation email. If our booking API is unavailable, your request will be sent via email using a secure mail server.

Security: All submissions must be validated on the server, protected using CSRF tokens and spam filters (reCAPTCHA v3 or honeypot as above).

What happens after you submit?

  1. Your details are securely sent to our appointment API (`POST /api/appointments`).
  2. Our team checks available slots with your preferred dentist and contacts you to confirm.
  3. On confirmation, you receive a WhatsApp and email with the appointment details and a calendar invite.

Backend & calendar integration notes

This section is for developers integrating the frontend with your backend and calendar or email system.

API endpoint: POST /api/appointments

Sample JSON payload:

{
  "name": "Jane Doe",
  "email": "jane@example.com",
  "phone": "+91XXXXXXXXXX",
  "service": "whitening",
  "preferredDate": "2025-01-15",
  "preferredTime": "16:00",
  "message": "Teeth whitening and check-up",
  "newsletterOptIn": true,
  "source": "web-form"
}

The API should return 201 Created with a JSON body containing an appointment ID, confirmed date/time (if auto-assigned) and any additional metadata.

Node/Express integration (example)

app.post('/api/appointments', csrfProtection, async (req, res) => {
  const { name, email, phone, service, preferredDate, preferredTime, message } = req.body;

  // TODO: Validate input on server as well
  // Save to DB
  const appointment = await db.appointments.insert({ ...req.body, status: 'pending' });

  // Send notification emails (e.g., using Nodemailer or SendGrid)
  await emailClient.send({
    to: 'reception@smilebrightdental.com',
    subject: `New appointment request from ${name}`,
    text: JSON.stringify(appointment, null, 2)
  });

  res.status(201).json({ id: appointment.id, appointment });
});
